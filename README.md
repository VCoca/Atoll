# Atoll Game with AI

Desktop application that allows users to play the Atoll board game
against another player or an AI opponent.

## Features

- Player vs Player mode
- Player vs AI mode
- Game state management and rule enforcement
- Graphical interface for gameplay

## Technologies

- Python
- Pygame

## How It Works

The game implements the rules of Atoll and manages the game state,
including valid moves and win conditions.

The AI opponent makes decisions based on the current game state
using a heuristic-based approach.

## Team Project

Developed in a team of three students, collaborating on game logic,
AI implementation, and user interface.

## What We Learned

This project helped us understand game development fundamentals,
state management, and basic AI techniques for decision-making.
We also gained experience working in a team and structuring a larger project.
